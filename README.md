CareConnect – Smart Elder Care Monitoring Platform
CareConnect is a web application that showcases an AI‑assisted, trustworthy home‑care monitoring experience for a single elderly patient. It focuses on caregiver accountability, medication safety, proactive alerts, and transparent communication for families.

Features
Smart dashboard showing patient status, vitals, activity level, visit timeline, and recent updates.

Visit history with reliability checks including late/missed visits, duration, task checklist, GPS/face‑match style verification, and anomaly summaries.

Medication & supplies panel with adherence, stock levels, and predictive “running low” alerts.

AI Anomaly Alerts center aggregating visit, medication, and health‑risk alerts with human‑readable explanations and suggested actions.

Family Portal with a Care Reliability / Trust Score, “today’s moments” feed, and quick actions (message nurse, request extra visit, start call).

Hospitals & Caretaker Requests page to request a nurse/caretaker from partner hospitals with a clear summary of the request.

Settings & accessibility including language, high‑contrast mode, font size, and routine coach toggles (hydration, light exercise, social check‑ins).
